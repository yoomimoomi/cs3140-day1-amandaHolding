### Setup Requirements (Node.js / npm)
You must be able to run JavaScript from your terminal.

#### Mac (recommended)
Install Node.js (best via Homebrew):

brew install node

Verify:

node -v

npm -v

#### Windows (required: Ubuntu via WSL)
Install WSL2 + Ubuntu

In Ubuntu terminal:

sudo apt update

sudo apt install -y nodejs npm

Verify:

node -v

npm -v

#### Linux (Ubuntu)
sudo apt update

sudo apt install -y nodejs npm

Verify with node -v and npm -v

### Terminal

Run: node hw1.js
